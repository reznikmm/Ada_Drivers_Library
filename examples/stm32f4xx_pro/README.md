The [STM32F4XX Pro](https://stm32-base.org/boards/STM32F407ZGT6-STM32F4XX-Pro) is
a low cost development kit for the STM32F407 micro-controller.
Sometimes, it can be purchased together with a 2.8" or 3.2" TFT LCD with a touchscreen.


You can get it here:

 - [Ebay](https://ebay.us/m/Kpgn8m)
 - [AliExpress](https://www.aliexpress.com/item/1005007067435793.html)
